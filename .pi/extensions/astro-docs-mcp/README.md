# Astro Docs MCP extension

Project-local pi extension that exposes the Astro Docs MCP server as a pi tool.

- MCP server: `https://mcp.docs.astro.build/mcp`
- Transport: Streamable HTTP
- Registered pi tool: `astro_docs_search`
- Registered command: `/astro-docs-mcp`

After adding or changing this extension, run `/reload` in pi or restart pi.
